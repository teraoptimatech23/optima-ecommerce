# Optima UI Guidelines

## Philosophy

The UI should feel:

- Premium
- Modern
- Minimalist
- Spacious
- Fast
- Mobile First

Inspired by:

- Nike
- Apple Store
- Shopify
- Linear
- Airbnb

Never create a cluttered interface.

Whitespace is preferred over crowded layouts.

Keep UI clean.

Use generous spacing.

Never overuse colors.

Primary action should always stand out.

Every screen should have only one visual focus.
